### 2. **Shopping Cart Program**
A shopping cart simulation with algorithm design, flowchart, and implementation.

**Features:**
- Algorithm documentation with pseudocode
- Visual flowchart representation
- Item selection with quantities
- Discount system (5% for total > R200)
- File-based receipt generation

### 3. **Student Scores Analyzer**
A program to manage and analyze scores for 5 students.

**Features:**
- Array-based score storage
- Calculation of average, highest, and lowest scores
- Score validation and input handling
- Clear score display and reporting

### 4. **Bank Account Management System**
A comprehensive banking system simulation with account management features.

**Features:**
- Account creation with validation
- Deposit and withdrawal operations
- Balance checking and account details
- Menu-driven interface
- Object-oriented design with functions
